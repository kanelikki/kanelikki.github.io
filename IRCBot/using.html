How to use

Setting

If you have no configuration file, it will ask some informations about the connection.

You can rewrite the settings by pressing c key when the program was just started.

Adding/Removing Modules

It's simple, just copy the module to Modules folder.

All modules are dll files. If the module is not proper, it won't be loaded.

NOTE: You can't add/remove modules at runtime. After adding or removing the module, you must restart the bot to apply it.

If you want to make your own module, of course you can do it easily.

Editing Configuration file

Warning!

Editing configuration file can cause error. If you feel that you ruined it, please remove the configuration file to reset. We recommend to backup the configuration file before editing.

If you want to edit just a little, you can edit IRCBot.exe.config file without typing everything from first.

You only need to change the value, but nothing else.
